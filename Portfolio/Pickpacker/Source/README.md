# Pickpacker Source Highlights

각 파일은 원본 프로젝트 구현과 대응하는 대표 코드입니다.

- `01_GameFlow_PickpackerGameMode.cpp`
  - 주문 웨이브, 주문 충족 판정, 크레딧 반영
- `02_Interaction_Inventory.cpp`
  - 상호작용 타게팅, 아이템 수집, 인벤토리 저장
- `03_AI_Architecture.cpp`
  - AI 공통 컨트롤러, 시야 설정, 패트롤 좌표 생성
- `04_Escape_EndingFlow.cpp`
  - 월드 플래그 기반 엔딩 조건 평가
- `05_MultiplayerSessions.cpp`
  - 세션 생성, 검색, 참가

원본은 아래 경로에서 확인할 수 있습니다.

- `Source/Blaster/`
- `Plugins/MultiplayerSessions/Source/MultiplayerSessions/`
