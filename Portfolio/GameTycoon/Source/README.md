# GameTycoon Source Highlights

각 파일은 `GameTycoon/src/` 쪽 구현과 대응하는 대표 코드입니다. (원본: `GameTycoon1207` 워크스페이스의 `GameTycoon/` 프로젝트)

- `01_GameBoard_GameLoop.java`
  - `GameBoard.java` — 메인 루프, 스크롤 뷰, 그리드 배치 클릭 처리
- `02_Developer_Movement.java`
  - `Developer.java` — 이동 스레드, 스프라이트 프레임, 목적지 이동
- `03_DevGame.java`
  - `DevGame.java` — 개발 진행도, 팀 스탯 반영
- `04_Game.java`
  - `Game.java` — 흥미도, 출시 가격, 판매 감쇠
- `05_Company.java`
  - `Company.java` — 개발 시작·진행·출시·일 매출, 세이브/로드, 경영 비용
- `06_GameUpdater_Events.java`
  - `GameUpdater.java` — 자금·날짜·진행 바 갱신, 개발 이벤트 탭
