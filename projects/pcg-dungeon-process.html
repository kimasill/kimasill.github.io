PCG 문서 기반 상세 구현 흐름(추가 예정).

Depth 재귀 로직

그리드 → 포인트 → FeedBack Loop로 Children 포인트 생성, Depth 만큼 반복하며 부모-자식 객체 구성.

구조 생성 전략

  • 층별 Unique Floor Interval 로 테마/변이도 부여
  • Cell Size 기반 정규화로 스케일 안정화
  • 계단: 높이 제약 + 경로 탐색으로 충돌 최소화

오브젝트 배치 최적화

  • Weight/Empty로 확률 제어 → 과밀도 방지
  • Clearance/Region 필터로 벽/방/복도 분리

향후 문서 확장

  • 프로파일링 스샷
  • 에디터 툴 UI 캡처
  • PCGEx 커스텀 노드 정리
