PCG Dungeon Generator (UE5 Plugin)

2개월 · 1인 개발 · 언리얼 PCG/PCGEx 기반 다층 던전 생성 & 재귀적 오브젝트 배치.

주요 기능

  • 다층 구조(Levels) 및 간격 제어
  • 벽/문/창문/통로/천장/기둥 자동 배치
  • 계단 모드(직선, ㄴ자, ㄷ자, Z자)
  • 가중치 & Seed 기반 오브젝트 랜덤 배치
  • Depth 기반 재귀적 자식 오브젝트 생성

설치

  1. Plugins 폴더에 복사
  2. UE 에디터에서 활성화 (PCG, PCGEx 필요 시 활성화)

메인 설정 (BP_Dungeon)

  • Level / Level Spacing
  • Cell Size / Minimum Cell Count / Grid Scale
  • Seed / Door Frame Size / Corner Size
  • Stair Options / Object Creation Depth

오브젝트 옵션

  • Placement Mode / Facing Mode
  • Weight / Empty / Clearance
  • Container / Children 재귀 생성
