Tab Game

Team: 1인

Project Period:2017.09 ~ 48시간

Platform: PC

Role: 기획, 프로그래밍, 미술/디자인

Engine & Tools: C# (.NET Windows Forms)

OVERVIEW

이 프로젝트는 클릭커 장르가 흔히 가지는 "클릭 → 성장 → 반복" 구조를 조금 더 상호작용적인 형태로 확장해보고자 하는 의도에서 시작되었습니다.

특히 단순 반복 클릭 중심의 플레이를 넘어, 마우스 클릭만으로 이루어지는 게임 구조 안에서도 플레이어가 상황을 판단하고 대응하는 요소를 만들고자 했습니다.

이를 위해 기존 클릭커 게임인 Tap Titans의 기본 구조를 참고하되, 플레이어를 공격하는 보스 몬스터와 페이즈 기반 전투 패턴을 도입했습니다. 저는 이 프로젝트에서 GUI 라이브러리를 통한 다양한 실험을 했고, 만족스러운 경험을 얻었습니다.
