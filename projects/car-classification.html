kCars Classification Module

국산 차량 데이터 셋 기반 분류 모델 (3개월 개발).

링크

GitHub Repo | 논문(PDF)

개요

차종/모델/연식 등 구분을 위한 커스텀 데이터 파이프라인과 증강(Augmentation) 전략을 적용한 모델.

기술

  • Python / PyTorch
  • 전처리: 이미지 정규화, 배경 제거
  • 증강: Random Crop, Color Jitter, Mixup
  • 모델: ResNet/ViT 실험 후 최종 앙상블

성과

  • 클래스 불균형 개선 (Weighted Sampler)
  • 모델 정확도 상승을 위한 앙상블 적용
