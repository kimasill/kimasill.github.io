소개

저는 게임 개발자이자 시스템 설계자입니다. 오랫동안 다양한 게임을 플레이해 온 게이머로서 무엇이 손맛을 만들고, 어떤 구조가 플레이어의 기억에 남는 경험을 만드는지 끊임없이 고민해 왔습니다. 그 고민을 실제 게임 시스템과 플레이 구조로 구현하는 것이 제가 가장 잘하는 일입니다.

Unreal Engine 5 기반 프로젝트에서는 협동 멀티플레이, AI 감시 시스템, 데이터 기반 게임플레이, 성능 최적화, 절차적 레벨 생성 플러그인까지 폭넓게 다뤄 왔습니다. 또한 Unity와 .NET 기반 MMORPG, Java Swing 기반 경영 시뮬레이션, Python 기반 딥러닝 분류 모델 등 서로 다른 환경에서도 시스템을 빠르게 이해하고 문제를 구조적으로 해결해 왔습니다.

저는 단순히 기능을 구현하는 개발자가 아니라, 기획 의도와 플레이 감각, 그리고 이를 안정적으로 뒷받침하는 기술 구조를 함께 설계하는 개발자입니다. 새로운 기술을 빠르게 학습하고, 병목과 충돌의 원인을 끝까지 파고들며, 다양한 직군과 협업 가능한 형태로 정리해 내는 것을 강점으로 삼고 있습니다.

핵심 역량

게임플레이 및 시스템

  • 게임플레이 프로그래밍, AI 설계, 상호작용 시스템 구현
  • 서버 권한 기반 멀티플레이 구조 및 동기화 설계
  • 데이터 에셋 기반 확장 가능한 시스템 설계
  • 월드 구조, 퀘스트 흐름, 경제 및 전투 시스템 기획

기술 문제 해결

  • UE5 프로파일링 기반 드로우 콜 및 렌더링 병목 개선
  • PCG와 C++ 커스텀 로직을 활용한 제작 파이프라인 구축
  • TaskQueue, Interest Management 등 구조적 안정성 설계
  • 데이터 편향 분석과 실험 기반 모델 성능 개선 경험

주요 프로젝트
Dawnstar
Unity / C# / .NET / 1인 / 6개월
2D 다크 판타지 MMORPG, 서버 구조부터 월드와 전투 설계까지 전 과정을 담당

Dawnstar는 세계관, 클라이언트, 서버, DB 구조, 경제 시스템, 퀘스트, 전투, 파티 플레이를 직접 설계한 1인 개발 MMORPG 프로젝트입니다. 인증 서버와 게임 서버를 분리하고, CommonDB를 기반으로 토큰과 상태를 관리하는 구조를 도입했으며, Protocol Buffers 기반 패킷 파이프라인으로 반복적인 직렬화 문제를 줄였습니다.

플레이어 주변 객체만 선택적으로 동기화하는 Interest Management 구조를 적용해 불필요한 네트워크 비용을 줄였고, 전투와 상호작용 로직은 룸 단위 TaskQueue 흐름으로 정리해 상태 충돌을 줄였습니다. 시스템 구현뿐 아니라 세계관 문서, 전투 리듬, 스킬-아이템 매핑 구조, 월드 탐험 흐름까지 함께 설계한 프로젝트입니다.

웹 페이지: https://kimasill.github.io/projects/dawnstar.html
개발 과정: https://kimasill.github.io/projects/dawnstar-process.html
Pickpacker
Unreal Engine 5 / 1인 / 개발 중
협동 멀티플레이 물류 시뮬레이션, AI 감시와 데이터 기반 게임플레이가 결합된 UE5 프로젝트

Pickpacker는 지하 세계 탐험과 물류 처리 루프를 결합한 협동 멀티플레이 게임입니다. 저는 게임플레이 프로그래밍, AI 설계, 멀티플레이 시스템, 포장 및 배치 구조를 모두 담당했습니다. 주문 태그, 레시피, 배치 규칙을 데이터 중심 구조로 정리해 신규 아이템과 목적지, 퀘스트 조건을 코드 수정 없이 확장할 수 있도록 설계했습니다.

또한 Mother AI 감시 시스템, 서버 권한 기반 상태 판정, OnRep 기반 IK 동기화, 자유 배치 및 스태킹 구조를 구현했고, 프로파일링을 통해 드로우 콜과 Scene Capture 병목을 찾아내 렌더링 비용을 개선했습니다. 실제로 캡처 갱신 구조와 인스턴싱을 정리한 뒤 프레임을 25FPS 수준에서 90FPS 이상까지 끌어올렸습니다.

개발 과정: https://kimasill.github.io/projects/pickpacker-process.html
PCGLevelGenerator
Unreal Engine 5.6+ / PCG / 1인 / 2개월
기획자와 아트 팀의 반복 작업을 줄이기 위한 UE5 절차적 레벨 생성 플러그인

이 프로젝트는 강한 랜덤성을 가진 던전 레벨을 빠르게 생성하기 위해 시작한 UE5 플러그인입니다. Grid 기반 그래프 구조와 MST 알고리즘을 활용해 방 연결을 보장하고, 다층 던전, 계단 규칙, 구조 생성, 오브젝트 배치, DataAsset 기반 설정 체계를 구축했습니다.

PCG 노드만으로 해결하기 어려운 수학적 규칙은 C++과 PCGEx 커스텀 로직으로 보완했습니다. 이를 통해 에디터 안에서 기획자나 아티스트가 파라미터를 수정하며 즉시 결과를 확인할 수 있는 제작 파이프라인을 만들었고, 테마 교체와 시드 변경만으로 다른 공간을 빠르게 생성할 수 있는 재사용 가능한 구조를 완성했습니다.
